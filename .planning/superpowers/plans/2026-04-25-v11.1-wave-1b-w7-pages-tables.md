## Background: the migration pattern (same as W4/W5)

For every `this.axios.<verb>(...)`:

1. Identify the URL → typed-client mapping.
2. Add `import { ... } from '@/api/<module>'`.
3. Replace `const response = await this.axios.get(url); const data = response.data;` with `const data = await typedHelper(...);`.
4. Drop unused `apiUrl` consts.
5. Drop the value-import of `axios` if it becomes unused. Type-only imports may remain.
