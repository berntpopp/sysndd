- [ ] **Step 4: Triage errors**

For each error type, decide:

- **Real bug** → fix (rare; if any are caught here, that's a win)
- **Type mismatch from missing annotation** → add the annotation

The goal is `npm run type-check:strict` returning green. If a particular file resists strict mode in a way that would balloon scope, narrow the include array to exclude that one file and add a follow-up issue.
