> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the three highest-leverage items left over from v11.1 — decompose `ManageUser.vue`, decompose `ModifyEntity.vue`, fix issue #29 entity-batch grouping — in a single PR fed by three parallel worktrees.

**Architecture:** Wave-1 runs three independent worktrees (W1 ManageUser, W2 ModifyEntity, W3 batch29) off a shared integration branch. Wave-2 closes out (version bump, review-doc update, AGENTS.md soft-LIMIT note, PR open).

**Spec:** `.planning/superpowers/specs/2026-04-25-v11.2-monolith-cleanup-design.md`

**Branch:** `feature/v11.2-monolith-cleanup` (created in M.1 below).
